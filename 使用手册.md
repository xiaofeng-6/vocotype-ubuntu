# VocoType CLI 使用手册

## 1. 安装 uv

若尚未安装 [uv](https://docs.astral.sh/uv/)，可先执行（官方推荐方式）：

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

将安装脚本提示的目录（常见为 `~/.local/bin`）加入 `PATH` 后，终端里应能直接运行 `uv` 命令。

## 2. 在本项目里建环境

```bash
cd ~/Desktop/Code/vocotype-ubuntu
uv python install 3.12    # 系统没有 3.12 时再执行，有则可省略
uv venv -p 3.12
source .venv/bin/activate
```

## 3. 装依赖、运行

```bash
uv pip install -r requirements.txt
python main.py
```

## 4. Linux 上需 root 才能用全局热键时

`keyboard` 在 Linux 上常要求 root。请在**项目根目录**、用 **当前 venv 里的 Python** 提升权限，不要用 `sudo python3`（会跑到系统解释器，没有 `keyboard` 等依赖）：

```bash
cd ~/Desktop/Code/vocotype-ubuntu
sudo -E .venv/bin/python main.py
```

`-E` 会尽量保留你当前环境变量，避免 PATH 与期望不一致。

在部分系统上，**用 root 跑时**桌面环境的 **PulseAudio / PipeWire** 会话仍可能拿不到麦克风，表现为「没有输入设备」。可尝试：保持 `sudo -E`、让 `XDG_RUNTIME_DIR` 等指向你的登录用户；或在项目里的 `config.json` 中设置 `audio.device` 为具体设备编号（在**普通用户**下执行 `python -c "import sounddevice as sd; print(sd.query_devices())"` 查看编号后写入配置）。
